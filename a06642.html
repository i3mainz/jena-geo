Detailed Description

Defines the requirements for an object that can be used as a tree node in a org.jdesktop.swingx.JXTreeTable.

This interface is used as a placeholder for the org.jdesktop.swingx.treetable.TreeTableNode interface defined in SwingX, in order to identify the code where we would use the later interface if we were allowed to introduce <cite>SwingX</cite> dependencies.

The first dependency to <cite>SwingX</cite> appears in the geotk-widgets-swing module, which provides an org.geotoolkit.gui.swing.TreeTableModelAdapter adapter} from this interface to the SwingX org.jdesktop.swingx.treetable.TreeTableModel tree table model}.
