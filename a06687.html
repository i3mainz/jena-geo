◆ interpolate1D()

double interpolate1D ( double  t0,
double  t,
double...  f 
)
protected

Cubic interpolation from 4 values.
With always t0 &lt= t&lt= t0 + 3

For example : cubic interpolation between 4 pixels.

   t =   0  1  2  3
f(t) = |f0|f1|f2|f3|
In this example t0 = 0.

Another example :
   t =  -5 -4 -3 -2
f(t) = |f0|f1|f2|f3|
In this example parameter t0 = -5.

Parameters
t0f(t0) = f[0].Current position from first pixel interpolation.
tposition of interpolation.
fpixel values from t = {0, 1, 2, 3}.
Returns
cubic interpolation at t position.
