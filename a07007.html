Detailed Description

Computes a set of arbitrary linear combinations of the bands of many rendered source images, using a specified matrix.

The matrix size (

numRows

×

numColumns

) must be equal to the following:

  • numRows
    : the number of desired destination bands.
  • numColumns
    : the total number of source bands (i.e. the sum of the number of source bands in all source images) plus one.

The number of source bands used to determine the matrix dimensions is given by the following code regardless of the type of ColorModel the sources have:

{ java int sourceBands = 0; for (int i=0; i<sources.length; i++) { sourceBands += sources[i].getSampleModel().getNumBands(); } }

The extra column in the matrix contains constant values each of which is added to the respective band of the destination. The transformation is therefore defined by the pseudocode:

{ java // s = source pixel (not all from the same source image) // d = destination pixel for (int i=0; i<destBands; i++) { d[i] = matrix[i][sourceBands]; for (int j=0; j<sourceBands; j++) { d[i] += matrix[i][j]*s[j]; } } }

In the special case where there is only one source, this method is equivalent to JAI's "{@link javax.media.jai.operator.BandCombineDescriptor BandCombine}" operation.
