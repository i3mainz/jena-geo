Detailed Description

Transforms vertical coordinates using coefficients from the Earth Gravitational Model.

See any of the following providers for a list of programmatic parameters:

  • org.geotoolkit.referencing.operation.provider.EllipsoidToGeoid

{

Note
This class is an adaption of Fortran code clenqt.for from the <cite>National Geospatial-Intelligence Agency</cite> and available in public domain. The <cite>normalized geopotential coefficients</cite> file bundled in this module is an adaptation of egm180.nor file, with some spaces trimmed.}
Author
Pierre Cardinal
Martin Desruisseaux (IRD, Geomatys)
Version
3.18
